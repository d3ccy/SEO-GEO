How to use this tool ▼

How to use

Enter a domain without http:// (e.g. numiko.com) and choose the location — this affects which Google index is used for keyword and competitor data.

Click Analyse Domain. Four API calls run in sequence — allow 10–20 seconds. Run for your client's domain first, then run for each of their top competitors to benchmark.

In the competitors table, click Analyse next to any competitor domain to run a Domain Overview on them directly.

Understanding the metrics

Domain Rank (0–100)
DataForSEO's authority score based on backlinks. 40+ is solid, 60+ is strong. A low rank with high traffic = content authority without backlink authority.
Est. Traffic / mo
Estimated monthly organic visitors from Google. Use as a relative benchmark — not an absolute traffic figure.
Positions 1–3
Keywords in the top 3 positions. These capture the majority of clicks for any given query.
Referring Domains
More meaningful than total backlinks — counts unique sites linking in. Diversity matters more than volume.
Dofollow vs Nofollow
Only dofollow links pass authority. A high nofollow proportion reduces the SEO value of the backlink profile.

Quick tips

  • Run before a project kickoff to establish a baseline — screenshot the metrics
  • Run for 2–3 competitors to benchmark where the client sits relative to the market
  • "Common Keywords" in the competitors table shows overlap — higher overlap = more direct competition
  • A high backlink count with a low referring domains count = most links from a few sites — less diverse = less authoritative
  • Sort the keywords table by Position to find quick-win keywords ranked 4–10 that could reach position 1–3 with optimisation

Common questions

Why are some sections empty?
Domain Overview uses several DataForSEO endpoints. Some API plans don't include all of them. Sections that fail show "not found" rather than blocking the rest.
The domain rank seems low — is something wrong?
Domain Rank is based on backlinks, not traffic. A site can have good organic traffic with a modest backlink profile. Use rank and traffic together for context.
Can I analyse a competitor's site?
Yes — enter their domain directly. Or click "Analyse" in the competitors table to jump straight to their overview.
