How to use this tool ▼

How to use

Enter any URL (with or without https://) and click Run Audit. No API key required — this tool fetches the page directly.

Optionally, select a Client Profile to auto-fill the URL field from saved details.

Once results appear, fill in the client name and click Download GEO Audit Report to generate a branded DOCX report you can share directly with clients.

Understanding the score

The 0–100 GEO Readiness Score measures how well a page is optimised for AI citation engines:

  • ● 71–100 — GEO Ready
  • ● 41–70 — Needs Work
  • ● 0–40 — Poor

The highest-value checks are JSON-LD Schema (+20pts) and AI Bot Access (+15pts). Fix these first.

What each check means

Page Title / Meta Description
"Too long" means the text exceeds 60/155 characters — AI engines may truncate it.
JSON-LD Schema
Structured data markup that tells AI engines what your content is about. Zero schemas = the biggest single gap.
AI Bot Access
Only scores if bots are explicitly allowed with Allow: / in robots.txt. "Not configured" is not the same as allowed.
Partial results banner
The site's WAF (e.g. Cloudflare) blocked the page. robots.txt and sitemap checks still ran.
Domain Authority panel
Backlinks data — shown for context only. Does not affect the score.

Quick tips

  • Audit before and after GEO changes to track improvement
  • If you see "Partial results — WAF blocked", ask the client to whitelist NumikoAuditBot/1.0 in their Cloudflare settings
  • Run on the homepage and 2–3 key service pages — scores often vary significantly
  • The DOCX report includes the full Princeton GEO methods reference — use it as a client deliverable

Common questions

Why is the score 0 even though the site has content?
The score only counts technical signals: title, description, schema, robots.txt, sitemap, and load time. Good prose does not affect the score.
The site loads in my browser but shows "Partial results" — why?
The audit server has a different IP from your browser. Some Cloudflare WAF rules block all server-side requests. Ask the client to allowlist the NumikoAuditBot user-agent.
Does the score include backlinks?
No. Backlinks are displayed for context but do not contribute to the 0–100 score.
