How to use this tool ▼

How to use

Fill in the Client name (required), domain, project name, and date. Choose the CMS — this customises section 3 of the guide so component names match the client's actual system.

Optionally upload a client logo (PNG or JPG, max 2MB) to replace the Numiko logo on the cover page. If not uploaded, the Numiko logo is used instead.

Load a Client Profile to auto-fill all fields. Click Generate Content Guide — the DOCX downloads immediately. No API key required.

What's in the document

Section 1 — Why This Matters
GEO explained in client-friendly language: how AI engines decide what to cite, and the business case for action.
Section 2 — The 5 Golden Rules
Princeton research methods with practical examples: cite sources, include statistics, use expert quotes, authoritative tone, plain English.
Section 3 — CMS Content Toolkit
How to use each content component (FAQ, Quote, Statistics callout, Key Fact, Rich Text) in the chosen CMS.
Section 4 — Page-by-Page Playbook
Checklists for About page, News/Blog, and Careers pages.
Sections 5–7
SEO and GEO checklist, before/after examples, and a print-ready A5 Quick Reference Card.

Quick tips

  • Generate one guide per client or project — the personalisation makes it feel bespoke
  • Share the DOCX with the client's content editors, not just the strategist — it's written for non-technical writers
  • The Quick Reference Card at the end can be printed A5 and pinned up in the editorial team's workspace
  • Use alongside the GEO Audit — audit first to identify gaps, then share the guide to explain how to fix them
  • Select "Other" from the CMS dropdown if the client uses a system not listed — generic component names will be used

Common questions

Can I generate a Content Guide without DataForSEO credentials?
Yes. The Content Guide is generated entirely within the app — no API calls are made. It works offline.
The file downloaded but won't open — what's wrong?
Ensure you have Microsoft Word, LibreOffice, or Google Docs. The file is a standard .docx format.
Can I customise the guide content?
The body content is fixed (based on Princeton GEO research). The cover page, client name, domain, CMS references, and logo are parameterised. For deeper customisation, ask the developer.
